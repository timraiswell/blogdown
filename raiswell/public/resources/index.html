Installing Python

Install Anaconda
Anaconda installs the latest version of the Python language (if you’re new to this, use version 3.X, not the obsolescent 2.7), several data science and analytic packages, and a few options for coding interfaces, including the popular (and recommended) Jupyter Notebooks. Jupyter is a browser-based interface for Python.
